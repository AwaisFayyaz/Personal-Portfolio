HOG-SVM Car Detection Pipeline on an Embedded GPU

The focus of this master thesis is on a vision-based car detection pipeline for a Cooperative Cruise Control system, based on Histograms of Oriented Gradients and Support Vector Machines. The pipeline is implemented on an automotive-grade embedded computation platform which features a Tegra K1 System on Chip, enclosing also a Graphics Processing Unit. The system is evaluated using the KITTI Vision Benchmark Suite, a standard in computer vision research. The results show that the pipeline reaches accuracies up to 65.5 percent, which is acceptable given the limited training dataset size of 1,174 samples and that application context has not yet been exploited
Supervisor: Prof. Dr. Peter de With, Video Coding and Architectures Group, Technical University of Eindhoven Netherlands
